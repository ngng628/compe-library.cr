class NgLib::SparseTable(T)

  • NgLib::SparseTable(T)
  • Reference
  • Object

Overview

不変な数列 $A$ について、以下の条件を満たす演算を、区間クエリとして処理します。

  • 結合則 : $(x \oplus y) \oplus z = x \oplus (y \oplus z)$
  • 冪等性 : $x \oplus x = x$

前計算は $O(N \log{N}) かかりますが、区間クエリには $O(1)$ で答えられます。
