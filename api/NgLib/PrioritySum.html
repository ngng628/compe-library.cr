class NgLib::PrioritySum(T)

  • NgLib::PrioritySum(T)
  • Reference
  • Object

Overview

昇順（降順） $k$ 個の総和を効率良く求めるためのデータ構造です。

値の追加、削除、$k$ の変更ができます。
