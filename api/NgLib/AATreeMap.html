class NgLib:: AATreeMap(K, V)

  • NgLib::AATreeMap(K, V)
  • Reference
  • Object

Overview

順序付き連想配列です。

平衡二分探索木として AA木 を使用しています。 性能は赤黒木の方が良いことが多い気がします。

C++の標準ライブラリの multiset と違って、$k$ 番目の値が取り出せることなどが魅力的です。
