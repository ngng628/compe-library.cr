class NgLib:: LDE(T)

  • NgLib::LDE(T)
  • Reference
  • Object

Overview

不定方程式 $ax + by = c$ を解きます。

常に解が存在するわけではないので、 #has_solution? などで解が存在するかを確かめるようにしてください。

多分、解の媒介変数 $m = 0$ のとき、$|x| + |y|$ が最小になります。
