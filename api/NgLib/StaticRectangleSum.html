class NgLib:: StaticRectangleSum(T)

  • NgLib::StaticRectangleSum(T)
  • Reference
  • Object

Overview

不変な二次元配列 $A$ に対して、$\sum_{i=l_i}^{r_i-1} \sum_{j=l_j}^{r_j-1} A_i$ を前計算 $O(N)$ クエリ $O(1)$ で求めます。
